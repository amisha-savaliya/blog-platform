# fullstack-blog-system_project
Project Overview

This is a full-stack blog application with user and admin roles, post engagement tracking, and secure authentication.

🔹 Features

User registration & login (JWT auth)

Admin & user role system

Create / edit / delete blog posts

Like & dislike system

Comment system

Post view counter

Search + pagination

Admin impersonation feature

🔹 Tech Stack

Frontend: React.js
Backend: Node.js, Express.js
Database: MySQL
Auth: JWT

🔹 How to Run

# Backend
npm install
node server.js

# Frontend
cd fronted
npm install
npm run dev

🔹 Database

Import blog.sql into MySQL.
